Install {{ app.name }}

In order to install {{ app.name }} with Software Manager,

  1. Install Flatpak and add {{ nuvola.name }} repository.
  2. Launch Software Manager.
  3. Search for {{ app.name }}.
  4. If there are multiple results, install a package from the stable branch. Unfortunately, this information is not available in the list of packages, but you need to visit package detail and wait a bit until this information is shown.
  5. {{ snippets.log_out_log_in(environments) }}

Alternatively, launch a Linux terminal emulator, type flatpak install nuvola {{ app.uid }} and press the Enter key to install this package.
