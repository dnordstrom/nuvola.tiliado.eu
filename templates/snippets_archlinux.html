Purge Old Nuvola Installations

If you used Nuvola in the past, it is necessary to purge the old installation. Otherwise, compatibility issues may occur.

You can use following commands to find installed Nuvola packages and then remove them with pacman -R. 

pacman -Qs '^nuvolaplayer'
pacman -Qs '^nuvolaruntime'
pacman -Qs '^nuvola-app-'

You can use following commands to remove old configuration.

rm -rf ~/.cache/nuvolaplayer3
rm -rf ~/.local/share/nuvolaplayer3 ~/.local/share/nuvolaruntime
rm -rf ~/.config/nuvolaplayer3 ~/.config/nuvolaruntime
rm -rf ~/.cache/nuvolaplayer3 ~/.cache/nuvolaruntime
rm -f ~/.local/share/applications/nuvolaplayer3*
